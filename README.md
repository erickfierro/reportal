## Installation

Descarga el programa [OpenCobolIDE](https://launchpad.net/cobcide/+download) disponible tanto para Windows, GNU/Linux y Mac OSX.

Instalar unicamente el compilador para Linux

```bash
# instalar gnucobol4
sudo apt-get install gnucobol4
```
